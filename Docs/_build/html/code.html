analyzer.assess(df, y_label, model_type='Linear', ranges=None)¶

(assess DataFrame str) This function takes in a dataframe of all the data, takes out the column addressed as y_label and then does a linear regression to use the x values to predict y_label and returns the stats (dict) and the predictions (list)

kargs:
model_type: (anyof ‘Linear’ ‘Logistic’ ‘Xgboost’ ‘SVM’ ‘Neural’) this determines which model will be used to assess the datapoints
ranges: (dictof col_name: (lstof min max)): This determines the full range of values that each column to take (used to normalize the data)
ex.
df_test = pd.DataFrame(data={‘x’:[1,2,3,4],’y’:[2,4,6,7]})
doRegression(df_test,’y’,model_type=’Linear’) -> {‘R^2’: 0.9796610169491525}, [[1, 2.2000000000000006], [2, 3.9000000000000004], [3, 5.6000000000000005], [4, 7.3]]

Note: Logistic defaults to one vs all classification

analyzer.plotBestFit(actual, predictions, filename=None)¶

(plotBestFit list list) Takes in the acutal datapoints and the predicted datapoints and then plots the two against each other. This will display the plot if filename is not provided.

kargs: filename: Saves the fig to this file

ex.
actual = [[1,2],[2,4],[3,6],[4,7]]
predictions = [[1,2.2],[2,3.9],[3,5.6],[4,7.3]]
plotBestFit(actual, predictions, filename=’./test.png’) -> A plt that shows a line of best fit and is saved to ./test.png
analyzer.plotPRC(prec, rec, labels, filepath)¶

(plotROC fpr tpr filepath) The purpose of this function is to create a plot depicting the ROC curve given the false positive rates (fpr), the true positive rates (tpr), and saving it to the filepath provided. >>> import numpy as np >>> from sklearn import metrics >>> y = np.array([1, 1, 2, 2]) >>> scores = np.array([0.1, 0.4, 0.35, 0.8]) >>> fpr, tpr, thresholds = metrics.roc_curve(y, scores, pos_label=2) >>> fpr array([0. , 0. , 0.5, 0.5, 1. ]) >>> tpr array([0. , 0.5, 0.5, 1. , 1. ]) >>> thresholds array([1.8 , 0.8 , 0.4 , 0.35, 0.1 ])
